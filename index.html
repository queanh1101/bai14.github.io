Lời giải:

  • c) Abadi – Phông chữ không chân (Sans-serif)

  • e) Berlin Sans FB – Phông chữ không chân (Sans-serif)
  • f) ALGERIAN – Phông chữ trang trí (Decorative)
  • g) Consolas – Phông chữ đơn cách (Monospace)
  • h) Cascadia – Phông chữ đơn cách (Monospace)

Luyện tập 2 trang 82

Đề bài: Hãy liệt kê các thuộc tính CSS liên quan đến định dạng đoạn văn bản sau.

Lời giải: Các thuộc tính CSS liên quan đến định dạng đoạn văn bản: căn đoạn văn bản, màu chữ, kiểu chữ. Cụ thể:

<!DOCTYPE html>

<html>

<head>

    <title>Định dạng đoạn văn bản</title>

    <style>

        .trangchu {

            font-weight: bold;

            color: blue;

            text-align: center;

            font-size: 24px;

        }

        .sangtac {

            text-align: right;

            font-weight: bold;

            color: red;

        }

        .cantrai {
            text-align: left;
        }

        .inngieng {

            font-style: italic;

        }

    </style>

</head>

Vận dụng 1 trang 82

Đề bài: Tìm hiểu thêm các thuộc tính phông chữ và dòng văn bản.

Lời giải:

  • font-variant: Xác định kiểu chữ viết hoa của văn bản.
    • normal: Chữ thường (mặc định).
    • small-caps: Chữ viết hoa nhỏ.
  • letter-spacing: Xác định khoảng cách giữa các kí tự. Ví dụ: letter-spacing: 2px;
  • word-spacing: Xác định khoảng cách giữa các từ. Ví dụ: word-spacing: 5px;
  • text-shadow: Tạo hiệu ứng bóng cho chữ. Ví dụ: text-shadow: 2px 2px 4px rgba(0,0,0,0.5);

Vận dụng 2 trang 82

Đề bài: Thiết lập hai tệp CSS khác nhau cho bài đọc thêm “Lịch sử CSS”.

Lời giải:

Kiểu 1:

/* style1.css */

body {

    font-family: Arial, sans-serif;

    font-size: 16px;

    color: #333;

}

h1 {

    font-family: "Times New Roman", serif;

    font-size: 28px;

    color: #990000;

}

p {

    font-family: "Courier New", monospace;

    font-size: 14px;

    color: #666666;

}

Kiểu 2:

/* style2.css */

body {

    font-family: Verdana, Arial, sans-serif;

    font-size: 18px;

    color: #555555;

}

h1 {

    font-family: "Helvetica Neue", Arial, sans-serif;

    font-size: 32px;

    color: #006600;

}

p {

    font-family: Georgia, serif;

    font-size: 16px;

    color: #888888;

}
